About the Workshop


Arduino is an open-source electronics platform based on easy-to-use hardware and software. Arduino boards are able to read inputs - light on a sensor, a finger on a button, or a Twitter message - and turn it into an output - activating a motor, turning on an LED, publishing something online. You can tell your board what to do by sending a set of instructions to the microcontroller on the board. To do so you use the Arduino programming language (based on Wiring), and the Arduino Software (IDE), based on Processing.

This workshop is aimed at introducing the students to the Arduino microcontroller and its basics. The students will use an Arduino UNO R3 board to learn about:

  • Embedded Systems
  • Specifications and Basics of Arduino
  • Digital I/O and Analog Signal Flow
  • Sensor Integration
  • Relay Interfacing
  • Home Automation
